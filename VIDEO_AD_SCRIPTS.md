## Production Checklist

### **Before Recording**
- [ ] Script memorized or on teleprompter
- [ ] Lighting set up (natural light or ring light)
- [ ] Microphone tested
- [ ] Background clean and professional
- [ ] Camera stable (tripod)
- [ ] Phone charged
- [ ] Quiet environment

### **Recording Tips**
- [ ] Record in 9:16 aspect ratio (mobile)
- [ ] Do multiple takes (5-10 per scene)
- [ ] Speak clearly and confidently
- [ ] Use hand gestures (natural)
- [ ] Make eye contact with camera
- [ ] Vary tone and pace
- [ ] Add energy and enthusiasm

### **Editing**
- [ ] Add captions (80% watch without sound)
- [ ] Use trending music (royalty-free)
- [ ] Add text overlays
- [ ] Include transitions
- [ ] Add sound effects
- [ ] Color grade for consistency
- [ ] Add CTA at the end
- [ ] Export in multiple formats

### **Distribution**
- [ ] YouTube (landscape + vertical)
- [ ] Instagram (vertical)
- [ ] TikTok (vertical)
- [ ] Facebook (vertical + landscape)
- [ ] LinkedIn (landscape)
- [ ] WhatsApp status (vertical)

## Video Specifications

### **YouTube**
- Format: MP4, H.264
- Resolution: 1920x1080 (16:9)
- Frame rate: 30fps
- Bitrate: 8-12 Mbps
- Duration: 60 seconds

### **Instagram/TikTok**
- Format: MP4, H.264
- Resolution: 1080x1920 (9:16)
- Frame rate: 30fps
- Bitrate: 5-8 Mbps
- Duration: 15-60 seconds

### **Facebook**
- Format: MP4, H.264
- Resolution: 1080x1920 (9:16) or 1920x1080 (16:9)
- Frame rate: 30fps
- Bitrate: 5-8 Mbps
- Duration: 15-60 seconds
